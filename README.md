# ToGo

> TO go interactive web app

## Requested packages to run project

``` bash
# install Vue
$ npm i vue -g

# Install NuxtJS
$ npm i nuxt -g

```

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
